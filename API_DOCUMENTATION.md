# API Documentation

## Authentication API


## Register User

Endpoint:

POST /api/auth/register


Example Request:

{
  "email": "admin@test.com",
  "password": "password123"
}


## Login User

Endpoint:

POST /api/auth/login


Example Request:

{
  "email": "admin@test.com",
  "password": "password123"
}


## Technology Stack

- ASP.NET Core Web API
- REST Architecture
- JWT Authentication
- Entity Framework Core
- BCrypt Password Hashing
